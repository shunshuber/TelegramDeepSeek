Привет, будущий программист!

Этот сайт поможет тебе изучить основы HTML и CSS. Здесь ты найдешь простые объяснения и веселые примеры!

Урок 1: Что такое HTML?

HTML - это язык, который помогает создавать структуру сайта.

HTML состоит из тегов, которые говорят браузеру, как отображать контент.

Например, тег <h1> создает заголовок, а тег <p> - абзац.

<h1>Мой первый сайт</h1>
<p>Привет, мир!</p>

Урок 2: Основные HTML-теги

Давай изучим самые важные теги, которые используются в HTML.

Вот некоторые основные теги:

  • <h1> - <h6>: заголовки разного уровня
  • <p>: параграф (абзац текста)
  • <a>: ссылка на другую страницу
  • <img>: изображение
  • <div>: контейнер для других элементов
<h2>Обо мне</h2>
<p>Меня зовут Аня, мне 9 лет.</p>
<img src="my_photo.jpg" alt="Мое фото">

Урок 3: Что такое CSS?

CSS - это язык, который делает HTML красивым!

С помощью CSS мы можем изменить цвета, шрифты, размеры и расположение элементов на странице.

CSS состоит из селекторов и свойств. Селектор выбирает элемент, а свойства определяют его внешний вид.

h1 {
  color: blue;
  font-size: 24px;
}

Этот код сделает все заголовки h1 синими и размером 24 пикселя.

Урок 4: Основные CSS-свойства

Изучим самые полезные CSS-свойства для оформления сайта.

Вот некоторые важные свойства CSS:

  • color: цвет текста
  • background-color: цвет фона
  • font-size: размер текста
  • margin: внешние отступы
  • padding: внутренние отступы
  • border: граница элемента
p {
  color: red;
  background-color: yellow;
  font-size: 18px;
  padding: 10px;
}

Этот код сделает все абзацы с красным текстом на желтом фоне с отступами.

Урок 5: Создаем свою первую страницу

Давай соберем все вместе и создадим простую веб-страницу!

1. Создай новый файл с расширением .html

2. Добавь базовую структуру HTML:

<!DOCTYPE html>
<html>
<head>
  <title>Мой сайт</title>
  <style>
    /* Твои стили CSS здесь */
  </style>
</head>
<body>
  <h1>Добро пожаловать!</h1>
  <p>Это моя первая страница.</p>
</body>
</html>

3. Добавь CSS в тег <style>, чтобы сделать страницу красивой!

4. Открой файл в браузере и посмотри на результат!
